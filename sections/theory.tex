\section*{Theory}
The decay process
\begin{align}
    \mu^+ \rightarrow e^+ + \nu_e + \Bar{\nu}_\mu
\end{align}
has a theoretical decay rate $\Gamma$. 

The Feynman diagram for the scattering process is
\begin{center}
    \feynmandiagram[layered layout,horizontal=a to b] {
        a [particle=\(\mu^{+}\)] -- [anti fermion] b -- [anti fermion] f1 [particle=\(\overline \nu_{\mu}\)],
        b -- [anti charged scalar, edge label'=\(W^{+}\)] c,
        c -- [anti fermion] f2 [particle=\(\nu_{e}\)],
        c -- [anti fermion] f3 [particle=\(e^{+}\)],
    }; 
\end{center}

We refer the statistical theory, which will comprise most of this report, to the Statistics subsection of the Results and Discussion Section.
